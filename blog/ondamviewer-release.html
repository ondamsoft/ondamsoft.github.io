OndamViewer 출시 - 가볍고 빠른 무료 이미지 뷰어

2026.02.20 · 5분 읽기

OndamSoft의 세 번째 제품, OndamViewer를 소개합니다. OndamZip(압축), OndamPDF(PDF 편집)에 이어 이미지 뷰어까지 — 무료 데스크톱 소프트웨어 라인업이 더 풍성해졌습니다.

왜 이미지 뷰어를 만들었나

Windows에 기본 내장된 '사진' 앱이 있지만, 실무에서 사용하기엔 아쉬운 점이 많습니다. 실행 속도가 느리고, 인쇄 옵션이 제한적이며, EXIF 정보 확인이 번거롭죠.

그렇다고 유명한 이미지 뷰어를 설치하면 광고가 따라오거나, 불필요한 기능이 잔뜩 포함되어 무거워지는 경우가 많습니다. "사진 한 장 보려고 왜 이렇게 복잡해야 하지?"라는 생각에서 OndamViewer 개발이 시작되었습니다.

주요 기능

다양한 이미지 포맷 지원

JPG, PNG, GIF, BMP, WebP, AVIF, TIFF, ICO, PSD 등 대부분의 이미지 포맷을 지원합니다. 특히 AVIF와 WebP 같은 차세대 포맷도 기본 지원하므로, 웹에서 다운로드한 이미지도 바로 열 수 있습니다.

직관적인 탐색

  • 마우스 휠로 같은 폴더의 이전/다음 이미지 탐색
  • 확대 상태에서는 마우스 휠로 줌 인/아웃
  • 가운데 휠 버튼을 누른 채 드래그하면 확대된 이미지를 자유롭게 이동
  • 가운데 휠 클릭만 하면 전체화면 토글

영역 선택 & 저장

이미지에서 원하는 영역을 드래그로 선택하고, 우클릭 메뉴에서 바로 저장할 수 있습니다. 저장 시 "복사-원본파일명.ext" 형식으로 기본 이름이 제안되며, 같은 이름이 이미 있으면 자동으로 번호가 붙습니다.

인쇄 기능

단순히 "인쇄" 버튼 하나가 아닙니다. 크기 설정(원본/페이지 맞춤/사용자 지정), 여백 및 위치 조정, 머리글/꼬리글 삽입까지 세밀한 인쇄 설정이 가능합니다. 사진을 인쇄할 일이 많은 분들에게 유용한 기능입니다.

EXIF 정보 표시

사진의 메타데이터를 한눈에 확인할 수 있습니다. 카메라 모델, 렌즈, 조리개, ISO, 촬영일 등의 정보를 사이드 패널에서 바로 볼 수 있습니다.

파일 연결 설정

설정에서 원하는 이미지 확장자를 체크하면 OndamViewer를 기본 이미지 프로그램으로 등록할 수 있습니다. 전체 선택/전체 해제가 가능하고, 확장자별로 개별 설정도 됩니다.

기술적 특징

OndamViewer는 Tauri 2.x(Rust 백엔드 + React 프론트엔드) 기반으로 개발되었습니다. Electron 기반 앱보다 설치 용량이 작고 메모리 사용량이 적습니다.

  • Rust: 이미지 디코딩, 파일 시스템 접근, 레지스트리 조작 등 성능이 중요한 부분을 처리
  • React 19: 직관적이고 반응형인 UI 구현
  • Shell Extension DLL: Windows 탐색기 우클릭 메뉴에서 바로 OndamViewer로 열기 가능

지원 플랫폼

  • Windows: Windows 10/11 64-bit (NSIS 설치파일)
  • macOS: macOS 12+ Apple Silicon (DMG, Apple 공증 완료)

무료, 광고 없음

OndamSoft의 모든 제품과 마찬가지로, OndamViewer도 개인/기업/기관 모두 무료입니다. 앱 내 광고도 없고, 유료 업그레이드 유도도 없습니다. 필요한 기능만 깔끔하게 담았습니다.

OndamSoft 4종 세트: OndamZip(압축) + OndamPDF(PDF) + OndamViewer(이미지) + OndamCapture(캡처)
모두 무료, 모두 광고 없음.
