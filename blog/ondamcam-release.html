OndamCAM 출시 - 무료 화면 녹화 프로그램

2026.03.23 · 5분 읽기

온담 패밀리의 다섯 번째 식구가 등장했습니다! 이번에는 화면 녹화입니다. 캡처를 넘어 움직이는 화면을 통째로 담아내는 OndamCAM을 소개합니다.

왜 화면 녹화 프로그램을 만들었나

업무 중 화면을 녹화해야 할 일이 의외로 많습니다. 버그 재현 영상, 사용법 안내 동영상, 온라인 강의 자료 등. 그런데 대부분의 녹화 프로그램은 유료이거나, 무료 버전에 워터마크가 박혀 있죠.

OndamCapture로 스크린샷은 해결했지만, "움직이는 화면도 찍을 수 있으면 좋겠다"는 요청이 많았습니다. 그래서 직관적이면서도 가볍고 완전 무료인 녹화 프로그램을 만들기로 했습니다.

주요 기능

세 가지 녹화 모드

  • 전체 화면 녹화 — 모니터 전체를 그대로 녹화. 멀티 모니터 환경에서 원하는 모니터 선택 가능
  • 영역 지정 녹화 — 화면 위에 투명 가이드를 띄우고, 원하는 영역만 지정하여 녹화. 가이드는 자유롭게 이동/크기 조절 가능
  • 창 녹화 — 특정 프로그램 창만 깔끔하게 녹화. 창 목록에서 선택하면 끝

시스템 오디오 + 마이크 녹음

화면만 녹화하면 반쪽짜리죠. OndamCAM은 시스템 오디오(스피커에서 나오는 소리)와 마이크를 동시에 녹음할 수 있습니다. 스테레오 믹스 설정 없이도 바로 동작합니다.

GPU 인코더 자동 감지

NVIDIA, Intel, AMD 그래픽카드의 하드웨어 인코더를 자동으로 감지하여 사용합니다. GPU 인코딩은 CPU 부담을 줄여주어, 녹화 중에도 PC가 느려지지 않습니다. GPU가 없으면 자동으로 CPU 인코더로 전환됩니다.

일시정지 & 재개

녹화 중 잠시 멈추고 싶을 때 일시정지 버튼을 누르면 됩니다. 불필요한 구간을 건너뛰고, 원하는 장면만 이어서 녹화할 수 있습니다.

직관적인 영역 가이드

영역 녹화 시 화면 위에 투명 오버레이 가이드가 표시됩니다. 파란 점선 테두리로 녹화 영역을 확인하고, 가장자리를 드래그하여 크기를 조절하고, 가운데를 잡고 이동할 수 있습니다. 녹화가 시작되면 가이드는 녹화에 찍히지 않습니다.

기술적 특징

OndamCAM도 온담 패밀리의 다른 제품들과 마찬가지로 Tauri 2.x(Rust + React) 기반입니다. 화면 캡처와 인코딩에 특화된 기술을 사용합니다.

  • xcap: 크로스 플랫폼 화면 캡처 라이브러리. 모니터/창 단위 캡처, DPI 스케일링 자동 처리
  • ffmpeg: 업계 표준 인코더. 캡처된 프레임을 실시간으로 H.264 MP4로 인코딩
  • cpal: 크로스 플랫폼 오디오 라이브러리. WASAPI loopback으로 시스템 오디오 캡처
  • WDA_EXCLUDEFROMCAPTURE: 오버레이 가이드가 녹화 영상에 찍히지 않도록 Windows API로 제외
