OndamCapture 출시 - 가볍고 빠른 무료 화면 캡처

2026.03.15 · 5분 읽기

온담 패밀리의 네 번째 식구, 온캡이가 태어났습니다! 삼색 고양이 온캡이와 함께하는 화면 캡처 프로그램 OndamCapture를 소개합니다.

왜 캡처 프로그램을 만들었나

Windows에는 캡처 도구(Snipping Tool)가 내장되어 있지만, 실무에서 사용하기엔 불편한 점이 있습니다. 단축키 설정이 제한적이고, 멀티 모니터 환경에서 원하는 모니터만 캡처하기가 번거롭죠.

기존 캡처 프로그램들은 기능은 많지만 무겁거나, 무료 버전에 광고가 따라오는 경우가 많습니다. "스크린샷 하나 찍는데 왜 이렇게 복잡해야 하지?"라는 생각에서 OndamCapture 개발이 시작되었습니다.

주요 기능

세 가지 캡처 모드

  • 전체 화면 캡처 (Ctrl+Alt+F) — 현재 화면 전체를 한 번에 캡처
  • 영역 선택 캡처 (Ctrl+Alt+A) — 원하는 영역만 드래그로 선택하여 캡처
  • 윈도우 캡처 (Ctrl+Alt+W) — 특정 창만 깔끔하게 캡처

글로벌 단축키

어떤 프로그램을 사용 중이든 단축키 한 번으로 바로 캡처할 수 있습니다. OndamCapture가 시스템 트레이에서 대기하고 있다가, 단축키를 누르면 즉시 캡처를 시작합니다.

멀티 모니터 지원

듀얼 모니터, 트리플 모니터 환경에서도 문제없이 동작합니다. 각 모니터별로 캡처하거나, 전체 화면을 통째로 캡처할 수 있습니다.

시스템 트레이 상주

OndamCapture는 시스템 트레이에 조용히 상주하면서, 필요할 때만 나타납니다. 메모리 사용량이 극히 적어서 PC 성능에 영향을 주지 않습니다.

기술적 특징

OndamCapture도 온담 패밀리의 다른 제품들과 마찬가지로 Tauri 2.x(Rust 백엔드 + React 프론트엔드) 기반으로 개발되었습니다.

  • Rust: 화면 캡처, 이미지 처리, 시스템 트레이 등 네이티브 기능을 빠르고 안정적으로 처리
  • React 19: 직관적인 캡처 UI 구현
  • 글로벌 단축키: OS 레벨에서 단축키를 등록하여 어디서든 즉시 캡처 가능

지원 플랫폼

  • Windows: Windows 10/11 64-bit (NSIS 설치파일)
  • macOS: macOS 12+ Apple Silicon (DMG, Apple 공증 완료)

무료, 광고 없음

OndamSoft의 모든 제품과 마찬가지로, OndamCapture도 개인/기업/기관 모두 무료입니다. 앱 내 광고도 없고, 유료 업그레이드 유도도 없습니다.
