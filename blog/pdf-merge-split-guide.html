PDF 합치기/나누기 방법 총정리 (무료)

2026.03.12 · PDF · 읽는 시간 약 7분

업무를 하다 보면 PDF 파일을 합치거나 나눠야 하는 상황이 자주 생깁니다. 여러 스캔 파일을 하나로 합쳐서 제출해야 하거나, 긴 보고서에서 특정 페이지만 뽑아야 할 때가 대표적입니다. 이 글에서는 PDF를 합치고 나누는 세 가지 방법을 비교하고, 각각의 장단점을 솔직하게 정리합니다.

왜 PDF 합치기/나누기가 필요할까?

PDF는 문서 교환의 사실상 표준이지만, 편집에는 약한 포맷입니다. 원래 "인쇄된 종이"를 디지털로 옮기려는 목적으로 설계되었기 때문입니다. 그래서 단순히 페이지를 합치거나 나누는 것도 전용 도구가 필요합니다. 흔히 겪는 상황은 이렇습니다.

  • 스캐너로 여러 장을 따로 스캔한 후 하나의 PDF로 합치기
  • 100페이지 보고서에서 3~5페이지만 추출하여 전달
  • 이력서, 자격증, 성적표를 하나의 파일로 병합하여 제출
  • 용량이 큰 PDF를 챕터별로 분할하여 공유
  • 계약서에서 서명 페이지만 별도 저장

방법 1: 데스크톱 프로그램 (OndamPDF)

보안이 중요하다면 이 방법

데스크톱 프로그램은 모든 작업이 내 컴퓨터 안에서 이루어집니다. 파일이 외부 서버로 전송되지 않으므로, 개인정보가 포함된 문서나 업무상 기밀 문서를 다룰 때 가장 안전한 방법입니다.

PDF 합치기 (병합)

  1. OndamPDF를 실행합니다.
  2. 병합할 PDF 파일들을 드래그 앤 드롭으로 추가합니다.
  3. 파일 순서를 원하는 대로 조정합니다.
  4. "병합" 버튼을 클릭하면 하나의 PDF로 합쳐집니다.
  5. 저장 위치를 선택하고 저장합니다.

PDF 나누기 (분할)

  1. 분할할 PDF 파일을 엽니다.
  2. 분할 모드에서 추출할 페이지 범위를 지정합니다. (예: 1-3, 5, 8-12)
  3. "분할" 버튼을 클릭합니다.
  4. 선택한 페이지가 별도의 PDF 파일로 저장됩니다.
  • 장점: 인터넷 불필요, 파일이 외부로 나가지 않음, 대용량 파일도 빠르게 처리, 무료
  • 단점: 프로그램 설치 필요

방법 2: 온라인 도구

편리하지만, 보안은 생각해 봐야 합니다

구글에서 "PDF 합치기"를 검색하면 수많은 온라인 서비스가 나옵니다. iLovePDF, Smallpdf, PDF24 등이 대표적입니다. 프로그램 설치 없이 웹 브라우저에서 바로 사용할 수 있어 편리합니다.

사용 방법 (대부분 비슷)

  1. 웹사이트에 접속합니다.
  2. PDF 파일을 업로드합니다.
  3. 합치기 또는 나누기 옵션을 선택합니다.
  4. 처리된 파일을 다운로드합니다.

보안 주의사항

온라인 도구를 사용할 때는 반드시 알아야 할 점이 있습니다. 내 파일이 외부 서버에 업로드된다는 것입니다. 대부분의 서비스는 일정 시간 후 파일을 삭제한다고 안내하지만, 실제로 삭제되었는지 확인할 방법은 없습니다.

따라서 다음과 같은 문서는 온라인 도구 사용을 피하는 것이 좋습니다.

  • 주민등록번호, 여권번호 등 개인 식별 정보가 포함된 문서
  • 회사 기밀 문서, 계약서
  • 의료 기록, 금융 정보
  • 아직 공개되지 않은 연구자료나 논문
  • 장점: 설치 불필요, 어떤 기기에서든 사용 가능, 간단한 작업에 빠름
  • 단점: 파일이 외부 서버로 전송됨, 용량 제한(무료 시 보통 25~100MB), 인터넷 필요, 무료 횟수 제한이 있는 경우 많음

방법 3: Adobe Acrobat

PDF의 원조, 하지만 비용이 부담

Adobe Acrobat은 PDF 포맷을 만든 Adobe의 공식 편집기입니다. 합치기, 나누기는 물론이고 텍스트 편집, 양식 작성, 전자 서명, OCR 등 PDF와 관련된 거의 모든 기능을 제공합니다.

문제는 가격입니다. Acrobat Pro는 월 구독 방식으로, 연간 계약 시 월 약 2만 원 수준입니다. PDF 합치기/나누기만 필요한 사용자에게는 과한 비용입니다. 무료 버전인 Acrobat Reader로는 합치기/나누기가 불가능합니다.

  • 장점: 가장 완성도 높은 PDF 편집 기능, 호환성 최고, 전문적인 편집 도구
  • 단점: 유료 (월 약 2만 원), 프로그램이 무거움(설치 크기 약 1GB), Reader로는 편집 불가

자주 묻는 질문

Q. PDF 합칠 때 순서를 바꿀 수 있나요?

네, 대부분의 도구에서 합치기 전에 파일 순서를 드래그로 조정할 수 있습니다. OndamPDF에서도 파일 목록에서 순서를 자유롭게 변경할 수 있습니다.

Q. 합친 PDF의 용량이 너무 커지면?

PDF 병합 자체는 용량을 크게 늘리지 않습니다. 만약 용량을 줄여야 한다면 별도의 PDF 압축(이미지 해상도 조절) 기능을 사용하세요. OndamPDF에서는 병합 후 바로 최적화(압축)를 적용할 수 있습니다.

Q. 스캔한 이미지도 PDF로 합칠 수 있나요?

JPG, PNG 등의 이미지 파일을 PDF로 변환하면서 합칠 수 있습니다. 이미지를 PDF로 변환하는 기능이 있는 프로그램을 사용하면 됩니다. 별도 변환 없이 이미지를 직접 병합할 수 있는 도구도 있습니다.

Q. 비밀번호가 걸린 PDF도 합칠 수 있나요?

PDF에 편집 제한 비밀번호가 걸려 있으면 합치기가 제한될 수 있습니다. 열기 비밀번호만 있는 경우에는 비밀번호를 입력한 후 합칠 수 있습니다. 비밀번호를 모르는 PDF의 편집 제한을 해제하는 것은 권장하지 않습니다.

결론

PDF 합치기/나누기는 어려운 작업이 아닙니다. 중요한 것은 보안과 편의성 사이에서 자신의 상황에 맞는 방법을 선택하는 것입니다.

  • 개인정보가 포함된 문서: 데스크톱 프로그램을 사용하세요. 파일이 내 컴퓨터를 벗어나지 않습니다.
  • 급하게 한두 번만 필요한 경우: 온라인 도구가 편리합니다. 단, 민감한 내용이 없는 문서에만 사용하세요.
  • PDF 편집이 주된 업무라면: Adobe Acrobat 구독을 고려하세요. 비용 대비 기능이 압도적입니다.
