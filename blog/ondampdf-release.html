OndamPDF 출시 - 무료 PDF 편집기의 새로운 선택

2025.12.10 · 약 7분 읽기

PDF 편집, 왜 이렇게 비싼가요?

PDF는 1993년 Adobe가 만든 문서 포맷입니다. 30년이 넘은 지금, PDF는 업무 문서의 사실상 표준이 되었습니다. 계약서, 이력서, 보고서, 전자책까지 PDF가 아닌 곳이 없습니다. 그런데 이상하게도, PDF를 제대로 편집하려면 여전히 비용을 내야 합니다.

Adobe Acrobat Pro는 월 23,100원입니다. 연간으로 치면 약 28만 원에 달합니다. 단순히 PDF 두 개를 합치거나, 페이지 순서를 바꾸거나, 워터마크를 넣는 작업에 이 비용이 합리적인지 의문이 드는 것이 당연합니다.

온라인 도구도 있지만 한계가 뚜렷합니다. 파일 크기 제한, 처리 속도, 그리고 가장 중요한 보안 문제가 있습니다. 회사 기밀 문서를 외부 서버에 업로드하는 것은 내부 보안 정책에 위반되는 경우가 대부분입니다.

OndamPDF는 이 문제를 해결하기 위해 만들어졌습니다. 완전 무료이면서, 모든 처리가 내 컴퓨터에서 이루어지는 데스크톱 PDF 편집기입니다.

OndamPDF 주요 기능

OndamPDF는 일상적인 PDF 작업에 필요한 핵심 기능을 모두 갖추고 있습니다.

PDF 보기

빠른 렌더링 엔진으로 수백 페이지 문서도 즉시 열립니다. 확대/축소, 페이지 이동, 북마크 탐색, 전체 화면 모드를 지원합니다. 다크 모드도 제공하여 야간 작업 시 눈의 피로를 줄여줍니다.

PDF 병합

여러 PDF 파일을 하나로 합칩니다. 드래그 앤 드롭으로 순서를 변경할 수 있고, 특정 페이지만 선택하여 병합하는 것도 가능합니다. 100개 이상의 파일도 한 번에 처리됩니다.

PDF 분할

하나의 PDF를 여러 파일로 나눕니다. 페이지 범위를 지정하거나, 일정 페이지 수마다 자동 분할하는 옵션을 제공합니다. 보고서에서 특정 챕터만 추출할 때 유용합니다.

이미지 변환

PDF를 PNG, JPG 등의 이미지로 변환하거나, 반대로 이미지 파일들을 PDF로 합칠 수 있습니다. 프레젠테이션 자료 제작이나 문서 공유 시 유용합니다.

워터마크

텍스트 또는 이미지 워터마크를 추가합니다. 위치, 크기, 투명도, 회전 각도를 세밀하게 조절할 수 있습니다. 대외비 문서에 "Confidential" 워터마크를 넣거나, 회사 로고를 삽입하는 용도로 사용됩니다.

보안 설정

PDF에 비밀번호를 설정하여 열기를 제한하거나, 인쇄/복사/편집 권한을 개별적으로 제어합니다. AES-256 암호화를 사용하므로 보안 수준이 높습니다.

인쇄 최적화

용지 크기에 맞게 자동 조절, 여러 페이지를 한 장에 인쇄하는 N-up 기능, 양면 인쇄 설정을 지원합니다.

Adobe Acrobat은 PDF 편집의 원조답게 텍스트 직접 편집, 양식 생성 등 고급 기능이 있습니다. 하지만 대부분의 사용자가 실제로 쓰는 기능은 보기, 병합, 분할, 인쇄 정도입니다. 이 핵심 기능만 놓고 보면 OndamPDF로 충분합니다.

왜 Tauri + React + Rust인가?

OndamPDF는 OndamZip과 같은 기술 스택으로 만들어졌습니다. Tauri 2.x 프레임워크 위에 Rust 백엔드와 React 프론트엔드를 결합했습니다.

Rust를 선택한 이유는 성능과 안전성입니다. PDF 파싱, 페이지 조작, 암호화 같은 작업은 CPU 집약적인데, Rust는 C/C++ 수준의 성능을 메모리 안전성과 함께 제공합니다. 실제로 100페이지 PDF 병합이 1초 이내에 완료됩니다.

Tauri는 Electron과 달리 Chromium을 내장하지 않고 OS의 WebView를 사용합니다. 그 결과 설치 파일이 약 30MB 수준으로, Electron 앱의 1/10 크기입니다. 메모리 사용량도 Electron 대비 60% 이상 적습니다.

React 프론트엔드는 직관적인 UI를 구현하는 데 집중했습니다. 복잡한 PDF 작업도 드래그 앤 드롭과 직관적인 버튼으로 쉽게 수행할 수 있습니다.

보안: 모든 처리는 내 컴퓨터에서

OndamPDF의 가장 중요한 특징은 모든 작업이 로컬에서 이루어진다는 점입니다. 파일이 외부 서버로 전송되지 않습니다.

온라인 PDF 도구를 사용하면, 여러분의 문서가 해외 서버에 업로드됩니다. 대부분의 서비스가 처리 후 파일을 삭제한다고 약속하지만, 전송 과정 자체가 보안 위험입니다. 회사 계약서, 의료 기록, 개인 신분증 사본 같은 민감한 문서를 외부 서버에 올리는 것은 피해야 합니다.

OndamPDF는 인터넷 연결 없이도 모든 기능이 작동합니다. 네트워크가 차단된 보안 환경에서도 사용할 수 있다는 뜻입니다. 이미지 변환, 워터마크 삽입 등 모든 처리가 로컬에서 이루어져 데이터가 외부로 나가지 않습니다.

사용 대상

OndamPDF는 다음과 같은 분들에게 적합합니다.

  • 직장인: 보고서 병합, 계약서 분할, 워터마크 추가 등 업무에서 빈번한 PDF 작업
  • 학생: 논문 자료 정리, 이미지→PDF 변환, 과제 PDF 병합
  • 프리랜서: 포트폴리오 제작, 인보이스 보안 설정, 클라이언트 문서 관리
  • 기업/기관: 대량 문서 처리, 보안이 중요한 내부 문서 작업 (완전 무료, 라이선스 비용 없음)

다운로드

OndamPDF는 Windows와 macOS를 모두 지원합니다.

  • Windows: OndamPDF 다운로드 페이지
  • macOS: Apple Silicon (M1/M2/M3/M4) DMG 제공

개인, 기업, 기관 모두 무료로 사용할 수 있습니다. 별도의 라이선스 구매나 구독이 필요 없습니다.
