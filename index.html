주요 기능

간결하지만 강력한 압축 도구

📦

11가지 포맷 지원

ZIP, 7Z, RAR, TAR, GZ, BZ2, XZ, ZST 등 주요 압축 포맷을 모두 지원합니다. 자체 포맷 ONDAM도 제공합니다.

🔒

AES-256 암호화

군사 수준의 AES-256 암호화로 파일을 안전하게 보호합니다. ZIP, 7Z, ONDAM 포맷에서 사용 가능합니다.

✂️

분할 압축

대용량 파일을 원하는 크기로 나누어 압축할 수 있습니다. 이메일 첨부나 이동식 저장 장치에 편리합니다.

⚡

초고속 압축

Rust 기반 고성능 엔진으로 기존 압축 프로그램 대비 월등히 빠른 압축/해제 속도를 제공합니다.

🖥️

크로스 플랫폼

Windows와 macOS를 모두 지원합니다. 어떤 환경에서든 동일한 사용 경험을 제공합니다.

🎨

직관적 UI

깔끔하고 현대적인 인터페이스로 누구나 쉽게 사용할 수 있습니다. 드래그 앤 드롭도 지원합니다.
