주요 기능

화면 녹화에 필요한 모든 기능

🖥️

전체 화면 녹화

모니터 전체를 고화질로 녹화. 멀티 모니터 환경에서 원하는 모니터 선택 가능.

✂️

영역 지정 녹화

투명 오버레이로 원하는 영역만 녹화. 드래그로 이동, 모서리로 크기 조절. 내부 프로그램 자유롭게 제어.

🪟

창 녹화

실행 중인 윈도우를 선택하여 해당 창만 녹화합니다.

🔊

시스템 오디오 녹음

스피커/헤드폰 소리를 그대로 녹음 (WASAPI Loopback). 스테레오 믹스 설정 불필요.

🎙️

마이크 녹음

마이크 입력을 함께 녹음. 시스템 오디오와 동시 녹음 가능. ON/OFF 즉시 전환.

⚡

GPU 인코더 자동 감지

NVIDIA NVENC, Intel QSV, AMD AMF 자동 감지. GPU가 없으면 CPU(libx264) 자동 전환.

⏸️

일시정지 / 재개

녹화 중 일시정지 후 이어서 녹화. 하나의 영상으로 자연스럽게 연결.

🔄

자동 업데이트

새 버전 자동 확인. 원클릭 업데이트. 항상 최신 기능 사용.

기술 스택

검증된 오픈소스 기반

Tauri 2.x React 19 xcap ffmpeg cpal (WASAPI) H.264 / MP4
