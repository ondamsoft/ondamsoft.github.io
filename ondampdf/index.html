주요 기능

PDF 작업에 필요한 모든 도구를 한 곳에

👁️

PDF 보기

빠르고 깔끔한 PDF 뷰어. 확대/축소, 페이지 이동, 검색 등 기본 기능 완비.

🔗

병합

여러 PDF 파일을 하나로 합치기. 드래그 앤 드롭으로 순서 변경.

✂️

분할

PDF를 페이지별로 나누기. 원하는 페이지만 추출 가능.

🔄

페이지 편집

페이지 회전, 삭제, 순서 변경. 직관적인 썸네일 UI.

🖼️

이미지 변환

PDF를 이미지로 변환하거나, 이미지를 PDF로 변환. 다양한 포맷 지원.

💧

워터마크

텍스트 또는 이미지 워터마크 삽입. 위치, 크기, 투명도 조절.

🔒

보안

PDF 암호 설정 및 해제. 인쇄/편집/복사 권한 제어.

🖨️

인쇄

소책자 인쇄, 모아찍기(N-up) 인쇄. 용지 절약 옵션.
