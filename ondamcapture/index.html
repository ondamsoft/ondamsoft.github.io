주요 기능

화면 캡처에 필요한 모든 기능

🖥️

전체 화면 캡처

모니터별 개별 캡처 또는 전체 모니터 합성 캡처. 멀티 모니터 완벽 지원.

✂️

영역 선택 캡처

드래그로 원하는 영역만 캡처. 점선 가이드와 실시간 크기 표시. 모니터를 넘나드는 선택 가능.

🪟

윈도우 캡처

실행 중인 윈도우 목록에서 선택하여 해당 창만 깔끔하게 캡처합니다.

⌨️

글로벌 단축키

Ctrl+Alt+F (전체), Ctrl+Alt+A (영역), Ctrl+Alt+W (윈도우). 어떤 앱에서든 즉시 캡처.

📋

클립보드 자동 복사

캡처 즉시 클립보드에 복사. Ctrl+V로 바로 붙여넣기 가능.

💾

자동 저장

캡처 시 자동으로 파일 저장. 저장 경로 설정 가능. 별도 저장 버튼도 제공.

🖼️

미리보기 & 편집

캡처 결과를 즉시 미리보기. 다시 캡처하거나 저장 전 확인할 수 있습니다.

🔽

시스템 트레이

트레이에서 조용히 대기. 창을 닫아도 백그라운드에서 단축키 대기합니다.

단축키

어디서든 바로 캡처

Ctrl+Alt+F 전체 화면 Ctrl+Alt+A 영역 선택 Ctrl+Alt+W 윈도우
